# Google Search Clone

A modern, responsive clone of Google's search interface built with HTML and CSS.

## Features

- **Google Search Page** - Classic Google homepage with search functionality
- **Image Search** - Google Images interface with image search capability
- **Advanced Search** - Fully functional advanced search page with multiple filter options
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Updated to match Google's current design 

## Pages

1. **index.html** - Main Google search page
2. **image.html** - Google Images search page
3. **advanced.html** - Advanced search with filters

## Technologies Used

- HTML5
- CSS3
- SVG Graphics
- Flexbox & Grid Layout
- Responsive Web Design

## Features Implemented

✅ Navigation bar with links to different pages  
✅ Google Apps icon  
✅ Sign in button  
✅ Interactive search bar with hover effects  
✅ Voice search icon  
✅ Google Lens (camera) icon  
✅ Search buttons  
✅ Footer with links  
✅ Advanced search form with multiple filter options  
✅ Fully responsive across all devices  

## How to Use

1. Open `index.html` in your web browser
2. Enter search terms and click "Google Search" to search on Google
3. Navigate between different pages using the navigation links
4. Try the advanced search page for more detailed search options

## Live Preview

Simply open any HTML file in your browser to see the clone in action!

## Responsive Breakpoints

- Desktop: Full featured layout
- Tablet (768px): Adjusted navigation and layout
- Mobile (480px): Stacked buttons and simplified navigation
